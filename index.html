¿Que es Css?

CSS significa Cascading Style Sheets (Hojas de Estilo en Cascada). Es un lenguaje usado para diseñar y dar estilo a las páginas web. Mientras HTML estructura el contenido, CSS se encarga de la apariencia, como los colores, las fuentes, el diseño, el tamaño de los elementos y cómo se distribuyen en la pantalla.

CSS es esencial en el desarrollo web porque permite diseñar páginas atractivas, adaptarlas a cualquier dispositivo, separar el contenido del diseño, garantizar consistencia, mejorar la velocidad de carga y añadir animaciones. Sin CSS, las páginas serían simples y sin estilo.

¿Que es JavaScrip?

JavaScript es un lenguaje de programación que permite agregar interactividad y dinamismo a las páginas web, como animaciones, manejo de eventos y comunicación con servidores. Es clave para que los sitios sean interactivos.

JavaScript es esencial en el desarrollo web porque permite agregar interactividad, realizar actualizaciones dinámicas, manejar eventos, crear animaciones y conectar con servidores para obtener o enviar datos. Sin JavaScript, los sitios web serían estáticos y poco interactivos.

Numero del atributo Nombre del atributo Descripción Valores posibles Ejemplo de uso Imagen representativa Fuente de información
1 font-size Define el tamaño del texto. Se puede especificar en unidades como px, %, em, rem, etc. 12px, 1.2em, 100%, large, small. font-size: 18px; w3schools.com
2 border-radius Redondea las esquinas de un elemento. Valores más altos hacen las esquinas más curvas. 5px, 50%, 10em, 0. border-radius: 10px; stackoverflow.com
3 display Controla cómo se muestran los elementos (bloque, en línea, flex, grid, etc.). block, inline, flex, grid, none. display: flex; w3schools.com
4 flexbox (display: flex;) Permite alinear y distribuir elementos en un contenedor flexible justify-content: center;, align-items: flex-start;, flex-direction: row;. display: flex; justify-content: center; align-items: flex-start; developer.mozilla.org
5 position Define cómo se posiciona un elemento (estática, relativa, absoluta, fija). static, relative, absolute, fixed, sticky. position: absolute; top: 50px; left: 100px; developer.mozilla.org
6 box-shadow Agrega sombras a un elemento, incluyendo color, desplazamiento y desenfoque. 5px 5px 10px rgba(0,0,0,0.5), none, 0px 4px 6px #aaa. box-shadow: 5px 5px 10px rgba(0,0,0,0.5); stackoverflow.com
7 overflow Controla cómo manejar el contenido que excede el área de un contenedor (ocultar, mostrar barras de desplazamiento, etc.). hidden, scroll, visible, auto. overflow: hidden; developer.mozilla.org
8 z-index Establece el orden de apilamiento de elementos, determinando qué está "encima" 1, 10, 999, -1. z-index: 999; stackoverflow.com
9 opacity Ajusta la transparencia de un elemento (0 = transparente, 1 = opaco). 0.5, 1, 0, 0.75. opacity: 0.7; developer.mozilla.org
10 margin y padding Margin ajusta el espacio exterior, y padding el espacio interior de un elemento. 10px, 2em, 5%, auto, 0. margin: 20px; padding: 10px; developer.mozilla.org
11 background-color Define el color de fondo de un elemento. #FF0000 (rojo), rgb(255,255,255) (blanco), transparent. background-color: #f3f3f3; developer.mozilla.org
12 text-align Controla la alineación del texto (izquierda, centrada, derecha, justificada). left, center, right, justify. text-align: center; developer.mozilla.org
13 height Establece la altura de un elemento en unidades absolutas o relativas. 100px, auto, 50%, inherit. height: 150px; developer.mozilla.org
14 width Define el ancho de un elemento con las mismas opciones de unidades. 50px, 100%, auto, inherit. width: 300px; developer.mozilla.org
